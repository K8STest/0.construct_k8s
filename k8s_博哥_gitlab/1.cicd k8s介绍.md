# 升级时候，不要跨版本升级

可以 V1.19.7 --> V1.19.8

如果是 V1.19. --> V1.20 # 需要查官方文档，对于 api version 的支持

https://github.com/kubernetes/kubernetes/blob/master/CHANGELOG/CHANGELOG-1.27.md#server-binaries-10

下载 kubernetes-server-linux-amd64.tar.gz 这个文件，解压缩按照 5.k8s 版本升级.png 进行操作

cd /etc/kubeasz

./ezctl -h
upgrade <cluster>

uname -m

如果输出 aarch64，表示是 ARM64 架构。
如果输出 x86_64，表示是 amd64 架构

```
kubectl config get-contexts
```
