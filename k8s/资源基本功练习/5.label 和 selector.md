# label 命令

```
kubectl label po <资源名称> app=hello


kubectl label po <资源名称> app=hello2 --overwrite

# selector 按照 label 单值查找节点
kubectl get po -A -l app=hello

# 查看所有节点的 labels
kubectl get po --show-labels


```

# selector 选择器

在各对象的配置 spec.selector 或其他可以写 selector 的属性中编写

kubectl get po -l type=app

kubectl get po -A -l type=app
增加 NAMESPACE 的显示

kubectl get po -A -l type=app --show-labels

kubectl get po -l 'test in (1.0.0, 1.1.0, 1.2.0)'

kubectl get po -l test!=1.1.0,type=app
and 的关系

kubectl get po -l 'test!=1.1.0,type=app,test in (1.0.0, 1.1.0, 1.2.0)'
