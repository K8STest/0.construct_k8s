# awk 命令

```

cat >> test.txt << EOF
shanghai+http://10.0.1.201+boge+devops1
beijing+http://10.0.1.202+boge+devops2
hongkong+http://10.0.1.203+boge+devops3
EOF

```

比方说 echo "213 ff" |tr ' ' '\n'命令，是可以直接输出内容的，但是 awk 不能直接输出结果
awk 是一个更复杂和功能强大的工具，主要用于模式匹配和文本处理。
在 awk 中，你可以执行各种操作（例如计算、条件判断、字符串处理等），这些操作不一定都要输出结果。因此，awk 默认不会输出任何东西，除非你明确告诉它输出什么。
echo "213 ff" | awk '{print $1}' ，默认会按照顾空格分隔

# awk 做计算

```
cat >> numbers.txt << EOF

10 20
30 40
50 60

EOF
```

awk '{sum=$1+$2; print "$sum"}' numbers.txt
在 awk 中，"$sum" 这个写法不会像在 Shell 脚本中那样进行变量替换。awk 使用的变量替换语法与 Shell 不同，因此 "${sum}" 这样的方式不会在 awk 中起作用。
awk '{sum=$1+$2; print sum}' numbers.txt # 需要这样写

awk '{if ($1 > 20) print $1}' numbers.txt

# # awk 字符串处理

```
cat >> strings.txt << EOF

hello_world
awk_script
bash_shell
EOF
```

awk -F\_ '{ print $1 }' strings.txt

# awk 使用内置函数

awk 提供了许多内置函数，如 length()、substr()、toupper()、tolower() 等。
awk '{print toupper($1)}' lowercase.txt

# awk 字符拼接

```
cat >> names.txt << EOF
> John doe
> Jane Smith
> EOF

 awk '{ print $1 ", " $2 }' names.txt
John, doe
Jane, Smith

```
