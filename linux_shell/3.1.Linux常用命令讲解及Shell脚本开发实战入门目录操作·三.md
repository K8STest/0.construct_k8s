# 这节课给大家带来的 linux 命令及 shell 脚本如下：

```
mkdir
	mkdir -p
touch                                     #修改·了文件属性·
	touch -d "1000 day ago" 1.log   或者  touch -d "2020-08-27" 1.log
stat
rm
	rm -rf
mv
	mv -f
cp
	cp -arf

```

# 我们来把这些命令串出来写个 shell 脚本运行一下，初期一定要多写，不要嫌简单，也不要嫌麻烦，当写到一定程度，工作中想到什么问题立马就能动手写出一个实用的 shell 脚本了：

```
#!/bin/bash

# 创建一个目录
mkdir mydir
# 创建一个空文件
touch myfile.txt
# 移动文件
mv myfile.txt mydir/
# 复制文件
cp mydir/myfile.txt mydir/myfile_copy.txt
# 看下这两个txt文件的状态
stat mydir/*.txt
# touch 改变下文件状态再查看有什么变化
touch -d "2020-08-27" mydir/myfile_copy.txt
stat mydir/myfile_copy.txt
# 删除目录及其内所有文件
rm -rf ./mydir

```
