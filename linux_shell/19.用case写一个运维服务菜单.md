这节课给大家使用 shell 里面的 case 写一个服务菜单，在有些运维选择的场景下，我们可能会需要使用到菜单来做选择，相较于 if elif else 而言，case 更适合用来做菜单类场景的脚本编写:

我们先用 `if elif else`

```
#!/bin/bash

# 这个脚本演示了如何使用 case 语句根据用户输入执行不同的操作

echo "请输入你的选择[1 - 4]:"
echo "1. 开始服务"
echo "2. 停止服务"
echo "3. 重启服务"
echo "4. 退出服务"

read choice

if [[ $choice -eq 1 ]];then
    echo "开始服务了"
elif [[ $choice -eq 2 ]];then
    echo "停止服务了"
elif [[ $choice -eq 3 ]];then
    echo "重启服务了"
else
    echo "退出服务了"
fi

```

下面使用`case`来写脚本：

```
echo "请输入你的选择[1 - 4]:"
echo "1. 开始服务"
echo "2. 停止服务"
echo "3. 重启服务"
echo "4. 退出服务"

read choice

case $choice in
    1)
        echo "Starting service..."
        # 执行启动操作的命令
        ;;
    2)
        echo "Stopping service..."
        # 执行停止操作的命令
        ;;
    3)
        echo "Restarting service..."
        # 执行重启操作的命令
        ;;
    4)
        echo "Exiting service..."
        exit
        ;;
    *)
        echo "Invalid choice: $choice"
        exit 1
esac
```
